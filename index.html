Key Features

Everything you need to manage university attendance efficiently.

QR-Code Attendance

Lecturers create daily session codes that students scan to sign attendance instantly — no logins required.

Geo-Location Verification

Optional GPS check ensures students are within a 500m radius when scanning for attendance.

Department-Level Dashboards

Heads of Department can view all lecturers’ attendance sheets and reports in one place.

Manual Attendance Entry

Lecturers can manually add names and IDs for students without phones directly into the system.

How It Works

1️⃣ Lecturer uploads class list and links to a course code.
2️⃣ Lecturer activates a QR code before class starts.
3️⃣ Students scan the code, enter ID and name.
4️⃣ LUNIX verifies entries (and location if enabled).
5️⃣ Attendance logs and absentees are automatically generated.
